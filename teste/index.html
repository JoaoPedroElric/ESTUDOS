NEGRITO / DESTAQUE

Para criar uma frase em destaque usamos a tag B. (Não semâmtica)
Com a tag b podemos marca partes importante que queremos dar mais destaque na frase.

Podemos usar STRONG (semântico) para destacar uma frase também.

ITÁLICO / ÊNFASE

Para colocarmos o texto em itálico utilizamos a tag I (Não semâmtoco)

Para deeixar o texto em ênfase de maneira semântica utilizamosa tag EM. (Semãntica)
